Introduction

What is CSS?

CSS, otherwise known as Cascading Style Sheets, is an external stylesheet used to design/customize a webpage (Stylesheet), however I would personally call it customizing style sheets.

Generally when learning CSS you would/should have a basic understanding coding languages such as,HTML.

Basic Info

When using CSS you would have an external file containing the CSS that you would have to link to the document/webpage(Stylesheet) with the command:

  • link rel="stylesheet" href="styles.css"

You would put this code withing your head tag at the start of a document

Syntax

CSS consists of a selector and a declaration block.

  • * { margin: 0; padding: 0; box-sizing: border-box; }

In the code above the * is the selector, the * selector selects all the code of the html document

Also in the code above the margin:0; is the declaration block, it declairs that the margin of the page for top, right,bottom, and left will be reset to 0.

Within the declaration block there consist of a property and value, the margin in the above code would be the property and the 0 of the above code would be the value.

More Syntax

Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces, for exampple:

  • body { font-family: Arial, sans-serif; line-height: 1.6; background-color: #f9f9f9; color: #333; padding: 20px; }
Selectors

CSS selectors are used to find/select the HTML elements you want to style.

There are five catagories for selectors:

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)
More Selectors

A quick example of all selectors:

  • Simple selector: * { color: red; }
  • Combinator selector: div p { color: red; }
  • Pseudo-class selector: a:visited { color: #00FF00; }
  • Pseudo-element selector: p::first-line { color: #ff0000; font-variant: small-caps; }
  • Attribute selector: a[target="_blank"] { background-color: yellow; }
Comments

CSS comments are not displayed in the browser, but they can help document your source code.

A CSS comment starts with /* and ends with */ these invalidate the code making the computer ignore them ex:

  • p { color: red; /* Set text color to red */ }

These comments can be made anywhere within the css file as long as it has /* */
